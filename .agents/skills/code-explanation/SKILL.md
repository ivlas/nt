# Code Explanation Skill

Explain `nt` code with this procedure:

1. Start with the command entry point.
2. Explain each module by responsibility.
3. Trace one command from parse to output.
4. Describe ownership and error handling where it matters.
5. Note where files are read or written.
6. Identify extension points and current constraints.
7. Keep the explanation concrete and tied to file paths.
